**ElectiGuide AI** is a high-resilience, multi-agent voter lifecycle management platform architected for the 2026 Hyderabad Election Cycle. It transitions the democratic experience from static information to a mission-critical "Command Center" interface, ensuring every voter is equipped with real-time intelligence, legal clarity, and heatwave-safe polling strategies.

### 🎨 The "Fig Mint" Aesthetic: Modern Heritage
ElectiGuide AI utilizes the proprietary **Fig Mint** design system—a "Modern Heritage" language that bridges the gap between 1970s hardware minimalism and 2026 technical precision.

- **Typography:** The authoritative elegance of `Instrument Serif` (Italic) paired with the terminal-grade clarity of `Geist Mono`.
- **Palette:** A high-contrast foundation of **Ink Black** (#1A1A1A) and **Warm Cream** (#F9F7F2), accented by **Fig Red** (#D32F2F) for machine intelligence and **Saffron** (#FF9933) for civic alerts.
- **Atmosphere:** Glassmorphism, subtle micro-animations via Framer Motion, and "Tear-Off" card physics that evoke the tactile feel of physical affidavits.

### 🛠 The Tech Stack
Built on the bleeding edge of the Next.js ecosystem to ensure total performance stability and E2E security.

- **Framework:** Next.js 16 (App Router / React 19)
- **Language:** TypeScript (Strict Mode)
- **Styling:** Tailwind CSS (Custom Design System)
- **Authentication:** Auth.js v5 (Passwordless OTP / JWT Strategy)
- **State Management:** Zustand (Multi-Stage Persistence)
- **Animation:** Framer Motion (Cubic-Bezier Physics)
- **Communications:** Resend (Real-Time SMTP Pipeline)

### 🤖 Multi-Agent Architecture
This platform was developed using a "Self-Healing" agentic loop, where specialized AI personas collaborated in real-time to harden the infrastructure:

- **@Security:** Enforced the move from mock providers to strict Auth.js credentials and hardened the `middleware.ts` against session bypasses.
- **@Backend:** Engineered the "Hard-Fail" API pipelines for Visual Crossing and NewsData.io, eliminating silent demo fallbacks in production mode.
- **@Frontend:** Implemented the Fig Mint Design System, ensuring the "Civic-Tech Noir" aesthetic remained cohesive across the entire voter journey.
- **@Tester:** Performed E2E validation of the "Golden Hour" weather algorithms and registration state persistence.

### 🏛 Core Features

#### 🛡 The Secure Gateway
A passwordless, high-security entry point utilizing **Auth.js v5**. Features a dual-channel OTP flow: real-time email delivery via **Resend** and terminal-logged dispatch for development redundancy. No voter data is persisted on the server, maintaining total anonymity.

#### 📋 Quest Registration
An editorial-style checklist that gamifies the bureaucratic process of voter registration. Implements "Tear-Off" card logic where completed steps are visually separated from the ledger, providing a satisfying sense of progression.

#### 📂 Candidate Dossier
A high-intelligence research hub providing unprecedented transparency into the 2026 candidates. Includes:
- **Wealth Transparency:** Automated Asset/Liability ratio visualizations.
- **Legal Audit:** Detailed criminal case histories and serious affidavit declarations.
- **Comparison Engine:** Side-by-side "Intelligence Briefings" for informed decision-making.

#### 🌡 Polling Day HQ
The mission control for the voter's actual journey to the booth. Integrated with the **Visual Crossing API** for real-time Hyderabad weather monitoring.
- **Golden Hour Algorithm:** Dynamically suggests optimal voting windows (e.g., 07:00 AM – 08:30 AM) if temperatures exceed 35°C, protecting vulnerable citizens from heatwaves.

#### 🤖 The Co-Pilot
A persistent, contextual AI assistant (Voter Assistant) built into the dashboard. It provides instant legal clarifications on election law and navigational help through the registration "Quests."

### 🧠 Prompt Engineering Gallery
The development of ElectiGuide AI utilized complex, multi-stage prompts to achieve its specific aesthetic and functional goals:

- **The Master Scaffolding Prompt:** Used to initialize the entire Next.js 16 architecture with predefined "Civic-Tech Noir" CSS variables and utility classes.
- **The Fig Mint Overhaul:** A comprehensive design-system-wide prompt that remapped legacy colors (Oxford Blue, Stamp Red) to the modern Fig Mint palette in a single pass.
- **The "Hard-Fail" Security Logic:** A directive that forced the AI agents to remove all demo fallbacks, ensuring the system would only run if live API keys were detected.
