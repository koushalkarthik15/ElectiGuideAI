# Brand Guidelines

**Theme**: "Civic-Tech Noir"

## Color Palette
- **Primary**: Deep Royal Blue (`#002366`) - Oxford Blue
- **Accent**: Burned Saffron (`#FF9933`)
- **Neutral**: Cream

## Typography
- **Headings (H1)**: Instrument Serif (Editorial Serif)
- **Body**: Inter
- **Data Points**: JetBrains Mono (Monospace)

## Design Elements
- **Aesthetic**: Editorial-Brutalist
- **Effects**: Glassmorphism constants for overlay panels and cards.
- **Interactions**: State-driven UI themes (e.g., transition from Blue to Red for alerts).
